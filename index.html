Geoselectie tool

Een paar snelle voorbeeldjes van functionaliteit voor een software component die in een formulier kan worden geintegreerd om een exacte locatie aan te geven. Beide voorbeelden gebruiken een reverse geocoding service om het dichtstbijzijnde adres op te zoeken. Het tweede voorbeeld gebruikt de geolocation api van de browser voor de start locatie.

Meer informatie is beschikbaar op de Geoselectie tool groep van Pleio.
